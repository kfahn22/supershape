SITE UNDER CONSTRUCTION

This website will show how to render 3D shapes in p5 using two different appproaches. The first is based on Daniel Shiffman's 3D supershape Coding Challenge and uses the supershape formula. The second uses shaders and is based on the work of Inigo Quilez and Martijn Steinrucken. You will see that you can obtain some very unique 3D shapes using both approaches!
