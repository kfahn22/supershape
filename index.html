This website contains the p5.js sketches I have written to render SUPER shapes in p5.js using several different approaches. The first several are based on Daniel Shiffman's 3D supershape Coding Challenge and use the supershape formula. The The mixture, rotation, and extrusion sketches use shaders and are based on the work of Inigo Quilez and Martijn Steinrucken. You will see that you can obtain some very unique 3D shapes using both approaches!
